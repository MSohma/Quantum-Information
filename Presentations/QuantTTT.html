$\ast$ algebra

We call a complex vector space $\mathfrak{A}$ satisfying the following properties $\ast$ algebra.
  • For any $A,B\in \mathfrak{A}$ we can define the product $AB\in\mathfrak{A}$:
    1. $A(BC)=(AB)C$
    2. $A(B+C)=AB+AC,(A+B)C=AC+BC$
    3. $\alpha(AB)=A(\alpha B)=(\alpha A)B,\alpha\in \mathbb{C}$
  • For any $A\in\mathfrak{A}$ we can define the involution $A^\ast\in\mathfrak{A}$:
    1. $(A^\ast)^\ast=A,(AB)^\ast=B^\ast A^\ast, B\in\mathfrak{A}$
    2. $(\alpha A+\beta B)^\ast=\bar{\alpha} A^\ast +\bar{\beta} B^\ast$,$\alpha,\beta\in\mathbb{C}$

Examples of $\ast$ algebra

  • $M_n(\mathbb{C})$:full matrix algebra of all $n\times n$ complex matrices.
  • $\mathfrak{B}(\mathfrak{H})$:algebra of bounded operators on a Hilbert space $\mathfrak{H}$
  • von Neumann algebra $\mathfrak{M}$: $\ast$ algebra $\mathfrak{M}\subset \mathfrak{B}(\mathfrak{H})$ satisfying $\mathfrak{M}^{\prime\prime}=\mathfrak{M}$, where $\mathfrak{M}^{\prime}=\{A\in \mathfrak{B}(\mathfrak{H})|AB=BA,\forall A\in\mathfrak{M}\}$ $\mathfrak{M}^{\prime\prime}=\{A\in \mathfrak{B}(\mathfrak{H})|AB=BA,\forall A\in\mathfrak{M}^{\prime}\}$
We can also consider $C^\ast$ algebra, which is a broader concept than von Neumann algebra.

Self-adjoint element

$\mathfrak{A}$:$\ast$ algebra
  • We call $A\in\mathfrak{A}$ satisfying $A=A^\ast$ a self-adjoint element.
  • The self-adoint elements represent observables in the quantum physics.
  • $\mathfrak{A}_S:=\{A\in\mathfrak{A};A=A^\ast \}$
    • 　
  • $\mathfrak{A}\ni A=A_{{re}}+iA_{{im}}$ $$A_{{re}}:=\frac{A+A^\ast}{2}\in\mathfrak{A}_S：\mbox{real part of }A$$ $$A_{{im}}:=\frac{A-A^\ast}{2i}\in\mathfrak{A}_S：\mbox{imaginary part of }A$$

State

$\mathfrak{A}$:$ast$ algebra with the identity $I$

A linear functional $\omega:\mathfrak{A}\to \mathbb{C}$ satisfying the following conditions is called a state.

  • $\omega(A^\ast A)\geq 0,\quad A\in \mathfrak{A}$
  • $\omega(I)=1$

When $\omega(A^{\ast}A)=0\Leftrightarrow A=0$, $\omega$ is called faithful.

It holds that $\omega(A^\ast)=\overline{\omega(A)}$.

Normal State

$\mathfrak{A}$ : von Neumann algebra

$\rho$ : density operator ($\rho\geq 0,\mbox{Tr}\rho=1$)

The state defined as

$$ \omega(A)=\mbox{Tr}\rho A $$ is called a normal state.
